## Atualizações da Semana

### Descrição

As ações realizadas durante a semana consistiram em realizar alguns ajustes solicitados pelas áres de projetos e negócios, as atividades em grande parte são relacionados a melhoria e a qualidade do produto Clube + STZ.

### 🆕 Implementações 
````markdown
➕ Suporte a TypeScript na matriz do código do Clube + STZ
➕ Modelo de acessibilidade para novos usuários
➕ Novos arquivos de configuração do Clube + STZ para desserialização
````

### 🔄 Atualizações 
````markdown
⬆️ Atualização de pacote do Bootstrap para o front-end
````

### ⚙️ Alterações 
````markdown
🛠 Ajuste na API responsável pelo envio de dados do formulário de cadastro do Clube + STZ
````
### 💥 Exclusões 

````markdown
✂ Pré-processador de CSS do site
✂ Protótipo não padronizado do Clube + STZ dos Modelos
````

### 🔒 Segurança 
````markdown
🆕 Implemnetação de login via oAuth2 (Clube + STZ)
🆕 Autenticação em 2 etapas '2FA' (Clube + STZ)
````
